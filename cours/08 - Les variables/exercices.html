Les variables

Pour répondre à ces exercices, tapez votre code dans le fichier exercices.js.
N'oubliez pas d'ouvrir la console développeur de Chrome pour voir vos erreurs JavaScript.

Vous êtes bloqué ? Demandez de l'aide sur Slack.

Objectifs

  • Comprendre les règles de nommage des variables en JavaScript.
  • Déclarer et utiliser des variables simples.
  • Comprendre et éviter les pièges concernant les variables.
  • Utiliser la console développeur de Chrome pour trouver la source des erreurs JavaScript.

Laissez toujours la console développeur ouverte pour ces exercices, vous pourrez y voir les erreurs JavaScript : Lorsqu'une erreur de syntaxe apparaît, tout le code s'arrête, par conséquent, les statuts des exercices deviennent tous "en attente de code à tester" tant que le bogue n'est pas corrigé.
Résolvez le bogue pour revalider tous vos exercices ! Bon courage ;-).
