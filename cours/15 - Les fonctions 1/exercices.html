Les fonctions (partie 1)

Pour répondre à ces exercices, tapez votre code dans le fichier exercices.js.
N'oubliez pas d'ouvrir la console développeur de Chrome pour voir vos erreurs JavaScript.

Consulter la doc MDN sur les fonctions

Vous êtes bloqué ? Demandez de l'aide sur Slack.

Objectifs

  • Comprendre le fonctionnement de fonctions simples (avec un seul paramètre).
  • Savoir déclarer et appeler une fonction.

Exercice 0

Tom est très fier de lui aujourd'hui, il a écrit sa première fonction !

Malheureusement, lorsqu'il lance son programme, rien ne se passe. Il est sûr que sa fonction est bien écrite, mais il ne comprend pas pourquoi le code ne s'exécute pas.

Aidez-le à faire en sorte que sa fonction soit exécutée.

Exercice 1

Hugo, le patron de Tom, veut automatiser l'écriture de l'en-tête des lettres qui seront envoyées pour la prochaine campagne marketing de l'agence MagiciensDuCode.

Voici le format de l'en-tête qu'il veut :

===[ Notre adresse ]===
MagiciensDuCode
42 avenue de l'accolade ouvrante
1337 POINT-VIRGULE

===[ Vos références ]===
N° de client : 420000

Tom doit rédiger une fonction afficherEntete(numeroClient) permettant d'afficher cet en-tête pour un numéro de client donné.

Il décide de découper cette tâche en 2 parties : la rédaction d'une fonction afficherTitre(titre) et d'une fonction afficherNumeroClient(numero).

Pour l'instant il se concentre sur la fonction afficherTitre(titre) qui permet d'afficher un titre au format ===[ titre ]===. Cette première version permettra d'afficher l'adresse de l'en-tête uniquement, sans la section "Vos références".

Il a déjà écrit une partie du code mais ne voit pas comment continuer. Aidez-le !

Exercice 2

L'affichage des titres et de l'adresse fonctionne bien. Maintenant Tom passe à la gestion du numéro client.

À cause de certaines règles de gestion client obscures, le format doit toujours être sur 6 chiffres et doit toujours commencer par 42.

Il doit donc créer une fonction afficherNumeroClient(numero) qui respectera cette règle. Il a décidé de tester son code avec les numéros client 0, 1, 9, 10, 99, 100, 999, 1000 et 9999. Ce code de test est déjà rédigé.

Il a fait la majorité du travail, mais le formatage sur 6 chiffres... il ne voit pas comment faire, aidez-le !

Exercice 3

Dernière étape, il faut maintenant que Tom utilise toutes ces fonctions dans le corps d'une ultime fonction afficherEntete(numeroClient) qui affichera l'intégralité de l'en-tête.

Par exemple afficherEntete(1337) devra afficher :

===[ Notre adresse ]===
MagiciensDuCode
42 avenue de l'accolade ouvrante
1337 POINT-VIRGULE

===[ Vos références ]===
N° de client : 421337

Tom essaye de s'appuyer sur toutes les fonctions et variables déjà écrites pour rédiger cette dernière fonction mais il a du mal à terminer.

Un petit coup de main de votre part ne serait pas de refus !
